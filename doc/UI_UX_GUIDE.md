# 🎨 숨표(Sumpyo) AI - 앱 UI/UX 디자인 가이드

모바일 기기에서 '아날로그의 포근함'과 '디지털의 편리함'을 전달하기 위한 디자인 원칙입니다.

## 1. 시각적 가이드라인 (Visual Guidelines)

### 컬러 팔레트 (App Color Palette)

| 이름              | 색상 코드 | 용도                            | 감성 효과          |
| ----------------- | --------- | ------------------------------- | ------------------ |
| **Warm White**    | `#F9F8F6` | 전체 배경, 종이 느낌            | 편안함, 깨끗함     |
| **Sage Green**    | `#B7C9B0` | 주 포인트 (약사 가운, 버튼)     | 안정감, 치유       |
| **Mute Blue**     | `#A1B5C1` | 보조 포인트 (청각 영양제, 헤더) | 차분함, 신뢰       |
| **Soft Charcoal** | `#4A4A4A` | 텍스트 (가독성 고려)            | 차분함, 고급스러움 |

### 타이포그래피 (Typography)

- **앱 UI (기본)**: `Pretendard` (모바일에 최적화된 높은 가독성)
- **감성 텍스트 (처방전)**: `나눔명조(Nanum Myeongjo)` 혹은 `고운바탕(Gowun Batang)` (따뜻한 아날로그 명조체)
- **강조 숫자 (고유 번호)**: `Montserrat` (세련된 고딕 계열)

## 2. 모바일 특화 인터랙션 (Reanimated Principles)

모든 움직임은 `React Native Reanimated`를 사용하여 60fps의 부드러움을 유지합니다.

### 핵심 애니메이션

- **심야의 약방 (Floating)**: 메인 화면의 약사 캐릭터가 위아래로 미세하게 떠 있는 듯한 움직임 (y: [-10, 10], duration: 4s).
- **조제 중 (Bouncing)**: 고민 입력 후 약사 캐릭터가 약을 짓는 귀여운 바운스 애니메이션.
- **처방전 공개 (Slide-up & Fade)**: 카드가 화면 하단에서 슬라이드 업되며 서서히 선명해지는 효과.
- **포인트 소모 (Shrinking)**: 결제 시 포인트 숫자가 살짝 작아졌다가 다시 돌아오는 피드백.

## 3. 사용자 여정 상세 (User Flow)

### Step 1: 고민 접수 (Consultation)

- **UI**: 은은한 전구색 조명이 비치는 종이 질감 배경.
- **UX**: "오늘 어떤 마음을 처방해 드릴까요?" 문구와 함께 부드러운 텍스트 박스 포커스.
- **디테일**: 키보드가 올라올 때 배경 요소들이 자연스럽게 위로 이동.

### Step 2: 스타일 선택 (Style Selection)

- **UI**: 3가지 테마(F/T/W)의 카드 아이콘 배열.
- **UX**: 각 스타일 선택 시 해당 페르소나의 목소리 톤을 반영한 짧은 대사 출력.
- **디테일**: 선택된 카드가 다른 카드보다 살짝 커지며 부드러운 그림자(Shadow) 효과 적용.

### Step 3: 결제 및 조제 (Payment & Alchemy)

- **UI**: "당신의 마음을 읽고 있어요..." 메시지와 함께 로딩 바 대신 약 짓는 애니메이션.
- **UX**: 결제 완료 시 짧고 기분 좋은 햅틱(Success) 진동 발생.

### Step 4: 처방 확인 (Prescription)

- **UI**: 화면 가득 차는 '디지털 처방전' 카드.
- **UX**: 상단 고유 번호부터 하단 음악 추천까지 차례대로 부드럽게 나타남.
- **디테일**: '인스타그램 스토리 공유'를 위한 카드 영역 최적화(9:16 비율 고려).

## 4. 리텐션 및 디테일 (Retention UX)

- **마음 일기**: 지난 처방전들을 날짜별로 모아보는 '나만의 약장' 그리드 뷰.
- **감정 분석 차트**: 사용자의 최근 감정 변화를 부드러운 꺾은선 그래프로 시각화.
- **푸시 알림**: "오늘 밤엔 일찍 자요"라는 짧은 '무료 비타민' 푸시 전송 시, 알림 클릭 시 즉시 차분한 배경음악과 함께 위로 문구 노출.
